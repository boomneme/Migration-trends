Project: Investigate a Dataset - International Migrant Stock

Introduction

Description

The data analyzed is obtained from Gapminder. This project visualizes migration patterns around the world by analyzing population data of foreign born individuals.

immigrant_n refers to the number of foreign born individuals in a country permanently.

emigrant_n refers to the number of individuals moving out of a country permanently.

Questions for Analysis

i. Which countries have the highest increase of population through migration?

ii. Which countries have the highest decrease of population through migration?

iii. Which country most reduced emigration over the last 10 years?

Data Cleaning

The data is astype objects. Since this is population data, it is converted to float, keeping in mind that values for k is 1,000 and M is 1,000,000.

In [113]:
def value_to_float(x):
    if type(x) == float or type(x) == int:
        return x
    if 'k' in x:
        if len(x) > 1:
            return float(x.replace('k', '')) * 1000
        return 1000.0
    if 'M' in x:
        if len(x) > 1:
            return float(x.replace('M', '')) * 1000000
        return 1000000.0
    return 0.0

mig_columns = ['1989', '1994', '1999', '2004', '2009', '2014', '2019']

Conclusion

From the analysis, the migration pattern of countries has shown the number of foreign born individuals in absolute values and the countries these individuals are leaving from.

Limitations

Exploration of the proportion that each country contributes to the other is not reflected.

This data can be further analyzed to define a relationships between the GDP of each country, conflict regions and type of governance to gain more insight to the causes of such migration patterns.
